## Requirement

- ทำ REST API โดยใช้ Node JS (JavaScript หรือ TypeScript) สำหรับโปรแกรม To do list โดยห้ามเชื่อมต่อกับ database
แล้วส่งงานผ่าน Github

## Installation

```bash
$ yarn install
```

## Running the app

```bash
$ yarn run start
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
